# Modern Image Gallery

A simple responsive image gallery with filters and lightbox functionality.

## Features
- Filter images by category (Nature, City, Animals, Tech, People)  
- Click an image to open lightbox  
- Navigate with Next / Prev buttons  
- Fully responsive design  

## How to Use
1. Open `index.html` in your browser.  
2. Click an image to view in lightbox.  
3. Use Next / Prev buttons to navigate.  
4. Use filter buttons to show specific categories.
